# Review and Merge

Review a PR and prepare it for merge with verification.

## Goal
PR reviewed, all issues addressed, tests green, and ready to merge (or clear merge conflicts).

## How to Run

```
/goal Review PR #456. Run tests, check for common issues, address feedback, and prepare the PR for merge. Use a verifier agent before final approval.
```

## Recommended Schedule

```
/loop 15m review-and-merge 456
```
